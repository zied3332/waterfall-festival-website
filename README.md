# Waterfall Festival Website

A modern, mobile-first website for Waterfall Festival Koh Phangan.

## Goal

Redesign and rebuild the public website from scratch to improve user experience, event visibility, ticket conversion, and brand presentation.

## Main Pages

- Home
- Events
- Event Details
- Tickets
- Gallery
- Experience
- Venue
- FAQ
- Contact
- AI Chat

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel

## Features

- Mobile-first design
- Responsive layout
- Countdown timer
- Event pages
- Ticket CTA linking to Eventpop
- Gallery
- FAQ
- Contact page
- AI assistant concept
- SEO optimized pages
